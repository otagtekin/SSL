NPM, Express, Nodemon

NPM
When we install node in our system, it comes with a built-in tool called, node package manager (NPM). This tool is design to install libraries and modules required for our code to run.
Here it is a sample npm command.

>> npm install express;

Express
Express is web application framework design for node. You can take a look at the documentation here. Express
This is where you start building MEAN (Mongo,Express, Angular,Node) stack apps. You don't have to use Angular since there are a lots of alternatives available today.

Once you install Express using the framework is pretty easy

var express=require('express'); // tell our application we will use express framework so bring the source
var app=express(); // tell our application that use the framework's features by referring to "app"
app.get('/',function(req,res) // tell framework that when someone types the root domain name or localhost use this action
{
  res.send('Hello SSL Students!'); // show users a message when the get("/") is called
});
var server=app.listen(3000,function() {}); // start the server by using the previous instructions.

Nodemon
Nodemon is service that monitors your node application and detects changes to your code. If there is change in the code it automaticsally restarts the server saving you from manually doing the work.

https://www.npmjs.com/package/nodemon

>> npm install nodemon;
global install npm install -g nodemon
to start server
nodemon app.js
or
nodemon server.js
