Let's setup our environment!

We are going to use Linux Ubuntu for this class. Your Mac already has terminal, it's always a good idea to practice workspaces that are designed for hosted code. Since it is not a good idea to use personal device to host a website, we will use a virtual machine with a virtual operating system to make things more realistic.

Things we need:

  • Terminal window
  • Code editor (I will use Sublime)
  • Virtual Box
  • Vagrant

Note:
Before you run Vagrant Up, you must enable the certain plugins. Open up your terminal and enter the following command before going through the next steps.

  vagrant plugin install vagrant-triggers

If your vagrant add box ubuntu/trusty64 is giving you " not found error" execute the following command and try again.

  sudo rm /opt/vagrant/embedded/bin/curl
