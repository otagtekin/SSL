Since it is not a good idea to use personal device to host a website, we will use a virtual machine with a virtual operating system to make things more realistic.

If you can execute your ES6 JavaScript because you are running Node version 10, that means you need to upgrade your node.

Check your node version by typing >>nodejs -v or node -v
After upgrade use node instead of nodejs
Here are the steps to upgrade your node in Ubuntu Linux
Stay in your virtual box
>>apt-get update
>>apt-get install build-essential libssl-dev
>>curl https://raw.githubusercontent.com/creationix/nvm/v0.30.2/install.sh | bash
>>nvm --version
>>nvm install node
>>node -v
